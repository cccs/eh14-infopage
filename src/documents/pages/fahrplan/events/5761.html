lecture: Mumble Workshop

Eine Einführung in Mumble (sowohl technisch als auch für Anwender).

Eine Einführung in die Open-Source Software Mumble.
Hierbei würde ich auf verschiedene Aspekte der Software eingehen, und sie in einem Resumee Konkurrenten wie Teamspeak/Ventrillo entgegenstellen. Dabei sollen alle 3 Aspekte (Anwender, Administrator, Hoster) zur Geltung kommen, und mit Beispielen sowie technischen Details versehen werden.
Einige technisch interessante Komponenten wären:

- Offen & frei (Lizenz)
- Sicher / verschlüsselt (SSL, Protokoll)
- Low-latency, man setzt auf OPUS
- Starke Integration des Nutzers (Voiceerkennung, positionsabhängiges Audio, Overlay, Skins)
- Erweiterbarkeit (ICE-Middleware)
- Administration ( ACL )
- Hosting ( Resourcen, Cluster-Hosting )

und schlußendlich auch das portable QT-Konzept zu erwähnen, welches sich z.B: für Web-Rendern lässt.
