lecture: BetterCrypto.org

Applied Crypto Hardening

Starke Kryptografie für Server korrekt konfiguriert. Ein Leitfaden für die Systemadministration.

Viele reden über starke Verschlüsselung und darüber, daß wir sie brauchen um uns gegen die globale Überwachung im Internet zu verteidigen. Wenn es denn nur so einfach wäre das auf den eigenen Servern korrekt zu konfigurieren. Es ist viel zu leicht gravierende Fehler zu machen.

Das BetterCrypto Projekt gibt eine Anleitung dazu wie gute Verschlüsselung einfach und vor allem korrekt implementiert werden kann.

Die Entstehung des Guides ist komplett transparent in einem Git Repository nachvollziehbar. Die gesamte Diskussion ist öffentlich. Alle Inhalte sind unter CC-BY zur freien Nutzung verfügbar. Alle Leute die zum Projekt beitragen tun dies ehrenamtlich.

Das Ziel ist den Überwachenden keinen Klartext mehr zu schenken!
